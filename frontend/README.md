# Frontend

Moderne Web-Oberfläche für das Mein Business Backend.

## 🚧 Status: In Planung

Das Frontend wird in Zukunft entwickelt.

## 📋 Geplante Features

- Dashboard mit Übersicht
- Mandanten-Verwaltung
- Rechnungserstellung (Web-UI)
- Finanz-Visualisierung
- Responsive Design

## 🛠️ Technologie-Stack (geplant)

- React / Next.js
- TypeScript
- TailwindCSS
- Chart.js für Visualisierungen

## 🚀 Setup (wenn entwickelt)

```powershell
# Virtuelle Umgebung (falls Python-basiert)
python -m venv venv
venv\Scripts\activate

# Oder für Node.js:
npm install
npm run dev
```
